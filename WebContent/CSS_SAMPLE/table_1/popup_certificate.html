1.교육대상

도움말

화면 개발 설명

  • 교육대상 위 내용은 화면이 뜨자마자 테이블1에서 읽어와야하며 JSTL 사용
  • 교육대상항목 필수
  • 체크박스 필수아님
  • 기타를 체크시 기타내용 필수 아님
  • 기타에 내용을 적고 다른 체크박스 클릭시 내용 초기화
    ex) 기타에 욕을 쓰지 마시오
  • 라디오버튼 필수, 건의사항 필수 아님
  • 체크박스는 DB에서 가져오기때문에 유동적으로 늘어남
  • 라디오버튼은 DB에서 가져오기때문에 유동적으로 늘어남
  • 여러분의 고민, 기타 건의 사항 최대 1000자 제한
  • 나이, 1+1항목 숫자만 적게
  • 저장 후 메시지 호출 하고 저장한 결과값을 보여주는 화면 호출
